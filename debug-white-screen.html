Quick Diagnosis Commands

Open the browser console on the white screen page and run these commands:

// 1. Check if React app is mounted console.log('Root element:', document.getElementById('root')); console.log('Body HTML:', document.body.innerHTML.substring(0, 200)); // 2. Check for render errors console.log('Any error boundaries triggered?', window.__REACT_ERROR_OVERLAY_GLOBAL_HOOK__); // 3. Check auth state const authState = JSON.parse(localStorage.getItem('situ8-user-store') || '{}'); console.log('Stored auth state:', authState); // 4. Check if services initialized console.log('Window services:', window.__SITU8_SERVICES__); // 5. Force clear auth and reload localStorage.removeItem('situ8-user-store'); localStorage.removeItem('situ8_auth_session'); location.reload();

Common White Screen Causes

  • ServiceProvider initialization failure
  • Circular dependency in imports
  • React error boundary catching fatal error
  • Infinite render loop
  • Module import failures

Debug Steps

  1. Open browser DevTools (F12)
  2. Check Console tab for red errors
  3. Check Network tab - are all JS files loading?
  4. Check Elements tab - is #root empty?
  5. Try hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
